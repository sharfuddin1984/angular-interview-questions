## ✨ Why this exists

Most "interview question" lists online are static walls of text — no search, no way to track progress, no way to focus on weak areas. This app fixes that:

- **Click to expand** — questions stay collapsed until you're ready to test yourself
- **Instant search** — filter across questions, answers, and code by keyword
- **Category filters** — jump straight to RxJS, Signals, NgRx, or any topic
- **Bookmarking** — star the questions you need to revisit, persisted locally
- **Copy-ready code examples** — one click to copy any code snippet
- **Fully responsive** — practice on your phone during a commute, no loss of functionality
- **Zero dependencies, zero backend** — a single HTML file, opens instantly anywhere

## 🚀 Getting started

No installation needed — it's a single self-contained HTML file.

```bash
git clone https://github.com/sharfuddin1984/angular-interview-questions.git
cd angular-interview-questions
open index.html   # or just double-click it
```

Or skip all that and use the **[live version](https://sharfuddin1984.github.io/angular-interview-questions/)** directly in your browser.

## 🛠️ Built with

Plain HTML, CSS, and JavaScript — no frameworks, no build step, no npm install. Just open it and it works. This was a deliberate choice: a prep tool for developers should never get in its own way with setup friction.
